# AlisverisSepetiArayuzu
Java Swing ve NetBeans kullanılarak tasarlanmış bir e-ticaret alışveriş sepeti arayüzü.

Apache NetBeans ve Java Swing kullanılarak tasarlanmış basit bir alışveriş sepeti masaüstü arayüzü (GUI) uygulaması. 

## Özellikler
* Ürün görselleri ve detaylarının listelenmesi.
* Sepetteki ürün adetlerini artırma/azaltma.
* Dinamik olarak güncellenen toplam sepet tutarı.
* Karanlık tema (Dark Mode) tasarımı.
